# MC_SVLootFilter

If you're mining, I suggest LadyHawk's loot filter.  It's much easier to configure and can be reconfigured on the fly.

F2 - enable/disable
F3 - lower filter level
F4 - raise filter level
F7 - toggles labels for filtered drops
